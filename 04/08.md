1）「收入」&「支出」类型切换组件

刚开始搞，就来整个最简单的！

在单文件组件里边，写 JS 区域的模板：

``` html
<script>
  export default { data, props, methods, created, /**...**/ }
</script>

// <=> 

<script lang="js">
  export default { data, props, methods, created, /**...**/ }
</script>
```
